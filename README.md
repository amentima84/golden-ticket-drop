# Golden Ticket Drop – Web & Cloud Task

This project demonstrates a simple full-stack application handling high-concurrency inventory updates without overselling.

## Tech Stack
- Backend: Java (Spring Boot)
- Frontend: React (Vite)

## How to Run

### Backend
```bash
cd backend
./mvnw spring-boot:run
The backend runs on http://localhost:8080

Frontend
bash
Copy code
cd frontend
npm install
npm run dev
The frontend runs on http://localhost:5173
